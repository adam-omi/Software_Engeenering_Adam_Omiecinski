# Simple Digital Thermometer 🌡️

Final Software Engineering laboratory project. Includes a working Python GUI application, complete IEEE 830 SRS documentation, and UML diagrams.

## 📂 Repository Contents

* **`thermometer.py`**: The final, executable Python application.
* **`Appendix_C_IEEE_DOCX/` & `Appendix_C_IEEE_PDF/`**: Contains all progressive versions (v1 to v5) of the Software Requirements Specification (SRS) in both Word and PDF formats.
* **`LAB_2/`, `LAB_3/`, `LAB_4/`**: Contains all UML diagrams (Use Case, Class, Activity) generated for the lab assignments.
* **`LAB_Presentations/`**: Original instructional materials and presentations provided during the course.

## 🚀 How to Run

The application uses Python's built-in `tkinter` library. No extra installation is required. Just run:

```bash
python thermometer.py
